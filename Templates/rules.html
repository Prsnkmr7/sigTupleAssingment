Diamond Game

Rules of the Game

  • A board of NxN with all cells coloured light grey.
  • Board size should be configurable based on the level of the game selected by the user on UI :

    Easy: 3X3

    Medium: 4X4

    Hard: 6X6

  • On game start, in every 1 second, a random cell is highlighted in green for 1 second.
  • If the user clicks the highlighted cell then score is incremented by 1.
  • If the user clicks any unhighlighted cell then score is decremented by 1. Score cannot go below 0..
